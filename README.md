## üöÄ Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
```bash
   npm install
```

2. Set your Gemini API key in `.env.local`:
```
   GEMINI_API_KEY=your_api_key_here
```

3. Start the development server:
```bash
   npm run dev
```
